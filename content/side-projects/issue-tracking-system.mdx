# Issue Tracking System

## Overview

A full-featured issue tracking system designed to help development teams manage bugs, features, and tasks throughout the software development lifecycle. Built with Laravel backend and React frontend, containerized with Docker for easy deployment.

## Key Features

- **Issue Management**: Create, update, and track issues with customizable statuses and priorities
- **User Roles**: Role-based access control for team members, managers, and administrators
- **Activity Tracking**: Comprehensive audit log of all issue changes and updates
- **Search & Filter**: Advanced filtering by status, priority, assignee, and custom fields
- **Responsive Design**: Mobile-friendly interface built with TailwindCSS
- **Docker Support**: Containerized application for consistent development and deployment

## Technical Highlights

- RESTful API built with Laravel for robust backend operations
- React SPA with TypeScript for type-safe frontend development
- MySQL database with optimized queries for performance
- Docker Compose setup for easy local development and production deployment
- Authentication and authorization with Laravel Sanctum
