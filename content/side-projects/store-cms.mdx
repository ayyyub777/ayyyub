# Store CMS

## Overview

A modern headless CMS built specifically for managing e-commerce stores. Provides a clean admin interface for managing products, categories, orders, and inventory while exposing RESTful APIs for frontend consumption.

## Key Features

- **Product Management**: Full CRUD operations for products with images, variants, and pricing
- **Category Organization**: Hierarchical category structure with nested subcategories
- **Inventory Tracking**: Real-time stock management and low-stock alerts
- **Order Management**: Process and track customer orders with status updates
- **Image Upload**: Integrated image handling with optimization
- **API-First Design**: RESTful API endpoints for easy frontend integration
- **Responsive Dashboard**: Modern admin interface built with React and TailwindCSS

## Technical Highlights

- Next.js 13+ with App Router for optimal performance
- Prisma ORM for type-safe database operations
- Server components for improved loading times
- API routes for backend operations
- Deployed on Vercel with automatic CI/CD

## Use Case

Perfect for decoupled e-commerce architectures where the frontend and backend are separate. Allows multiple storefronts to consume the same product data via APIs.
